# Shahrisabz Tibbiyot Texnikumi

Premium landing page for Shahrisabz Tibbiyot Texnikumi — a modern medical education institution website.

## Tech Stack

- Next.js 15 (App Router)
- React 19 + TypeScript
- Tailwind CSS v4
- Framer Motion + GSAP
- Lucide Icons
- Embla Carousel
- Yet Another React Lightbox
- Leaflet Maps

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Configuration

Update contact details and map coordinates in [`lib/site-config.ts`](lib/site-config.ts).

Replace the placeholder logo at [`public/logo.svg`](public/logo.svg).

## Build

```bash
npm run build
npm start
```
